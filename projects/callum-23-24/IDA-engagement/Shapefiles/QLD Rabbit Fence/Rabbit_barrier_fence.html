Description

Abstract / Description:
The Queensland rabbit fence 2016 is a linear feature class that describes the location of the rabbit fence maintained by the Darling Downs - Morton Rabbit Board. It should be viewed in conjunction with the wild dog barrier and check fences 2014. The dataset was last updated in 2016.

History:

Lineage statement:
The linear feature class was originally created in November 2008 using Garmin GPS points (accuracy 5-10m) collected by Biosecurity Officers in the (then) Department of Primary Industries. The linear features have been updated using ortho-rectified 10 to 30cm aerial imagery captured in 2013.

The feature class was edited in ArcGIS desktop 10.3.
It contains the following attributes:
OBJECTID: Internal feature number.
Shape: Feature geometry.
TOPNET: Describes whether the fence has a topnet (Y = yes, N = no)
LG1: Local Government name to north and east of fence (sourced from Department of Resources).
LG2: Local Government name to south and west of fence (sourced from Department of Resources).
Shape_Length: Length of feature in internal units.
FEATURE: Name of the feature (rabbit fence).
Effective legislation date: Date when the fence was legislated.
Edited date: Date when the linear feature was edited.
Length in km: Length of fence or fence segment, in kilometers. Calculated in Australian Albers94, using ArcGIS calculate geometry function.
