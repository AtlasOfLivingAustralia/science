Description

Abstract / Description:
Queensland wild dog barrier and check fences is a linear feature class that describes the location of the wild dog barrier and check fences.
It should be viewed in conjunction with the Rabbit Barrier Fence.
The fence was most recently updated in 2016. The dataset is described in the Biosecurity Act 2014, and requests to change the dataset must follow the procedure as described in the Act.
Purpose:
The dataset is described under the Biosecurity Act 2014.

History:

Lineage statement:
The dog fence was originally digitized by the Biosecurity Queensland wild dog fence maintenance team using Garmin GPS (spatial accuracy of 5 to 10m). The location of the fence was further refined in June 2016 using ortho-rectified aerial photographs or SPOT imagery captured in 2012 - 2013.

The feature class was edited in ArcGIS desktop 10.3.
It contains the following attributes:
OBJECTID: Internal feature number.
Shape: Feature geometry.
Shape_Length: Length of feature in internal units.
FEATURE: Name of the feature type, either Wild dog barrier fence or Wild dog check fence.
Effective legislation date: Date when the fence was legislated.
Edited date: Date when the linear feature was edited.
Length in km: Length of fence or fence segment, in kilometers. Calculated in Australian Albers94, using ArcGIS calculate geometry function.
